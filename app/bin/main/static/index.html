📖 របៀបសរសេរ

SabayLang ជាភាសាកម្មវិធីដែលប្រើពាក្យគន្លឹះភាសាខ្មែរ ដើម្បីសរសេរកូដ។

🔢 ប្រភេទទិន្ន័យ

តាង ចំនួន ក = 10 // ចំនួនគត់ តាង ចំនួន ខ = 3.14 // ចំនួនទស តាង គ = "សួស្តី" // តួអក្សរ

🔢 ប្រមាណវិធី

+ បូក - ដក * គុណ / ចែក

⚖️ ប្រៀបធៀប

== ស្មើ != មិនស្មើ > ធំជាង < តូចជាង >= ធំឬស្មើ <= តូចឬស្មើ

📚 មេរៀន

១. ការប្រកាសអថេរ

តាង ចំនួន ឈ្មោះ = តំលៃ;

ឧទាហរណ៍: តាង ចំនួន អាយុ = 20។

២. ការបង្ហាញលទ្ធផល

ហៅ "សារ";

ប្រើ ហៅ ដើម្បីបង្ហាញអត្ថបទ ឬ តំលៃ។

៣. លក្ខណៈ If/Else

បើសិន លក្ខណៈ { ហៅ "True"; } ផ្សេង { ហៅ "False"; }

៤. Else If

បើសិន ក > 10 { ... } ម្យ៉ាងវិញ បើសិន ក > 5 { ... } ផ្សេង { ... }

💡 ចំណាំសំខាន់

⚠️ ច្បាប់សរសេរ

  • បន្ទាប់ពីប្រកាសអថេរ ត្រូវប្រើ ។ (។ ជាសញ្ញាបញ្ចប់)
  • ប្រើ {} សម្រាប់ block កូដ
  • ប្រើ " សម្រាប់ string
  • អថេរត្រូវមាន ឈ្មោះ ពីរខ្ទង់ (ឧ: ក, ខ, គ)

🔤 ពាក្យគន្លឹះ

តាង - ប្រកាសអថេរ ចំនួន - ប្រភេទចំនួន ហៅ - បង្ហាញ បើសិន - ប្រសិន ម្យ៉ាងវិញ - ប្រសិនផ្សេង ផ្សេង - ផ្សេង
